# ClipPictureLib
模仿微信图片裁切，基于github开源社区代码。
添加了对图片exif信息的读取，预处理旋转角度来适配三星Samsung系列手机自动旋转照片的问题
添加图片压缩和模糊算法

20161215 提交使用demo
